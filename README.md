## ✨ Fitur

- **Preloader animasi** — loading screen dengan progress bar & status network
- **Custom cursor** — kursor dengan efek trail
- **Interactive Globe** — globe 3D interaktif berbasis Canvas, draggable, dengan node skill & project
- **Typed.js** — teks berganti otomatis di hero section
- **Smooth scroll & reveal** — elemen muncul saat scroll dengan animasi fade-up
- **Active nav highlight** — nav link berubah warna sesuai section yang sedang aktif
- **Mobile nav** — hamburger menu dengan overlay backdrop dan animasi slide
- **WhatsApp widget** — tombol floating WhatsApp untuk kontak langsung
- **Dark theme** — full dark mode dengan palette `#050507`
- **Fully responsive** — mobile, tablet, dan desktop

## 🚀 Cara Menjalankan

**Tanpa framework — cukup buka langsung:**

```bash
# Clone repo
git clone https://github.com/far7code/far7code.github.io.git

# Masuk folder
cd far7code.github.io

# Buka di browser (atau gunakan Live Server di VS Code)
open index.html
```

> Disarankan menggunakan [Live Server](https://marketplace.visualstudio.com/items?itemName=ritwickdey.LiveServer) di VS Code agar font & asset lokal termuat dengan benar.
